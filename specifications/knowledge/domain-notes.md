# Домен: оренда/продаж житла (Київ) — нотатки

Факти й нюанси, які агент має знати, працюючи над ботом.

## Валюта — ключове
- **Оренда** зазвичай у **грн**. **Продаж** — майже завжди у **$** (іноді €), НЕ в грн.
- Тому будь-який парсер ціни мусить розрізняти валюту й нормалізувати в грн. **Ніколи** не підписувати
  «сире число» іноземної ціни як грн (це помилка на ~×40).
- Не змішувати «повну ціну» з «ціною за м²» (обидві бувають поруч у розмітці).

## DOM.RIA (офіційне developer API, developers.ria.com)
- 2 кроки: `GET /dom/search?api_key=…&category=1&operation_type=3&state_id=10&city_id=10&lang_id=4`
  → `{count, items:[id…]}` (newest-first); `GET /dom/info/{id}?api_key=…&lang_id=4` → деталі.
- **operation_type:** `3` = довгострокова оренда, `1` = продаж. **Київ:** `state_id=10`, `city_id=10`.
- **Валюта:** `currency_type_id` 1=USD, 2=EUR, 3=UAH. `priceArr` містить ту саму ціну в усіх трьох;
  ключ **`"3"` = завжди грн** (за поточним курсом) — беремо його.
- **Фільтр ціни:** `price_from`/`price_to` фільтрує по **грн-еквіваленту** (конвертує $/€) — можна
  безпечно передавати гривневий поріг. Площа/кімнати в публічному API **не** фільтруються.
- **Фото CDN:** `https://cdn.riastatic.com/photos/<path>` з розмірним суфіксом `b` перед `.jpg`
  (не `photosnew/…` — той 404).
- Ліміти: пакетний ключ (напр. 20000/міс). Free-tier мав годинний cap (`HourOverlimit` 429).

## Rieltor (rieltor.ua, server-rendered HTML)
- Каталог: `/flats-rent/` (оренда), `/flats-sale/` (продаж). URL-фільтри (перевірені): `price_min`,
  `price_max`, `rooms=N` (точні 1–3), `f-owners=1` (лише власники).
- Картка: `.catalog-card` з `data-catalog-item-id` (id), `data-label` (ціна: «20 000 грн» для оренди,
  «185 000 $» для продажу), `.catalog-card-details` (кімнати «N кімнати», площа «80 / 60 / 10 м²» —
  беремо перше число), `.catalog-card-region` (район «…р-н»), `.catalog-card-author-subtitle`
  («Рієлтор» | «Власник»). Посилання: `/flats-*/view/{id}/`. Фото: CDN `market-images.lunstatic.net`.
- **Sale-ціна в грн:** у `title` атрибуті `.catalog-card-price-title` є НБУ-еквівалент —
  «По курсу НБУ - 8 306 389грн / 88 366 грн/м²» → перше «X грн» = повна ціна в грн.
- Клас-хеші CSS-модулів дрейфують при редеплої фронту — покладатись на `data-*` атрибути й ld+json, де є.

## Доступність джерел із дроплета (дата-центровий IP)
- **DOM.RIA, Rieltor** — доступні. **OLX, lun.ua, flatfy.ua** — Cloudflare `403` (потрібен резидентський
  проксі). **birdrent, josti** — застосунки без веб-каталогу (скрапити нема що). Див. [ADR-0003](../architecture/decisions/0003-sources-landscape.md).

## Пастки (footguns)
- **`REDIS_KEY_PREFIX` дефолт `olx`** — це неймспейс БД (спадок старої назви), **не** джерело. Ключі —
  `olx:profile:*`, `olx:seen:*`. Перейменування = **міграція даних** (осиротить усі профілі/seen), не env-правка.
- Ціни без значення → оголошення **виключається** (не показуємо «договірну»).
- Майстер — **лише Київ**; ціна голим числом = «до» (максимум); незрозумілий ввід → перепит.
